# MiniTools

A collection of simple, useful, and interactive web tools built with HTML, CSS, and JavaScript.

MiniTools is a lightweight utility website that brings multiple everyday tools into one place. The project focuses on creating clean interfaces, practical functionality, and an easy user experience without relying on external frameworks.

## 🌐 Live Demo

Try MiniTools here:

**https://9minitools.netlify.app/**

## Features

MiniTools currently includes:

* **Calculator**
  Perform basic mathematical calculations quickly.

* **Age Calculator**
  Calculate your exact age based on your date of birth.

* **BMI Calculator**
  Check your Body Mass Index using your height and weight.

* **Unit Converter**
  Convert between different units easily.

* **Number Guessing Game**
  A simple interactive guessing game.

* **Password Generator**
  Generate secure random passwords.

* **Quote Generator**
  Generate random quotes.

* **Text Counter**
  Count words and characters from text.

* **Tip Calculator**
  Calculate tips and split bills easily.
